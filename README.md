Бизнес-центр имеет ограниченное количество парковочных мест (например, 5). Посетители приезжают в разное время. Если все места заняты, машина ожидает освобождения одного из мест, но не дольше чем 5 секунд. Как только место освобождается, одна из ожидающих машин занимает его Парковка не может быть перегружена — нельзя въехать, если нет свободного места.
